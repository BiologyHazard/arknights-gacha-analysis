4. 抽卡系统解析
    1. 达成目标所需寻访次数的分布
        1. 任意寻访
            1. 在任意寻访中获得若干名任意6★干员
            2. 在任意寻访中获得若干名任意5★及以上干员
            3. 在任意寻访中获得若干名任意5★干员
        2. 常驻标准寻访
            1. 在常驻标准寻访中获得若干名任意UP6★干员
            2. 在常驻标准寻访中获得若干名指定的UP6★干员
            3. 在常驻标准寻访中获得全部2名UP6★干员各至少若干次
            4. 在常驻标准寻访中获得若干名任意UP5★干员
            5. 在常驻标准寻访中获得若干名指定的UP5★干员（需验证保底情况）
            6. 在常驻标准寻访中获得全部3名UP5★干员各至少若干次（需验证保底情况）
        3. 限时标准寻访（活动单UP卡池）
            1. 在限时标准寻访中获得若干名UP6★干员
            2. 在限时标准寻访中获得若干名指定的UP5★干员
            3. 在限时标准寻访中获得UP6★干员和指定的UP5★干员各至少若干次
        4. 联合行动
            1. 在联合行动中获得若干名任意UP6★干员（等价于在任意寻访中获得若干名任意6★干员）
            2. 在联合行动中获得若干名指定的UP6★干员（保底情况未知）
            3. 在联合行动中获得若干名指定的UP5★干员（保底情况未知）
        5. 前路回响
            1. 在前路回响中获得若干名任意UP6★干员（等价于在任意寻访中获得若干名任意6★干员）
            2. 在前路回响中获得若干名指定的UP6★干员（保底情况未知）
        6. 定向甄选
            1. 在定向甄选中获得全部UP6★干员各至少若干次
        7. 限定寻访
            1. 在限定寻访中获得若干名任意UP6★干员（考虑/不考虑寻访数据契约兑换）
            2. 在限定寻访中获得若干名指定的UP6★干员（考虑/不考虑寻访数据契约兑换）
            3. 在限定寻访中获得全部2名UP6★干员各至少若干次（考虑/不考虑寻访数据契约兑换）
            4. 在限定寻访中获得若干名UP5★干员（考虑/不考虑寻访数据契约兑换）
            5. 在限定寻访中获得全部2名UP6★干员和UP5★干员各至少若干次（考虑/不考虑寻访数据契约兑换）
            6. 在限定寻访中获得若干名指定的5倍出率权重提升的UP6★干员（考虑/不考虑寻访数据契约兑换）
        8. 联动寻访
            1. 在联动寻访中获得若干名UP6★干员
            2. 在联动寻访中获得若干名任意UP5★干员
            3. 在联动寻访中获得若干名指定UP5★干员
            4. 在联动寻访中获得全部2名UP5★干员各至少若干次
            5. 在联动寻访中获得UP6★干员和全部2名UP5★干员各至少若干次
    2. 平稳分布（综合出率）
        1. 任意寻访6★、5★、4★、3★干员综合出率
        2. 若限时标准寻访获得UP6★干员就停手，此时的6★、5★、4★、3★干员综合出率
    3. 平均每次寻访能获得的凭证数量
        1. 非中坚寻访平均每次能获得的资质凭证、高级凭证数量
        2. 若限时标准寻访获得UP6★干员就停手，此时每次能获得的资质凭证、高级凭证数量
        2. 中坚寻访平均每次能获得的通用凭证数量
    4. 其他内容
        1. 获得UP6★干员时，获得的其他6★干员数量、5★干员数量的分布
        2. 干员池十连获得6★、5★、4★、3★干员数量的联合分布和边际分布
        3. 干员池十连总星数的分布
5. 引用、致谢
